Clarification on the usage of Flexbox

By: Alexa Dela Cruz
Hi! Lately, I have been using "display: flex;" on most of my divs as I thought that this would allow me to put multiple elements inside my divs. However, as time progresses, I realized that this just makes my elements float (it does not move the way I want to, even with the use of margins!), which messes up some of my divs. Do you have any recommendations on when "display: flex;" would be needed?

Francheska Vicente
This has been my problem until a while ago, and my (current, I might be wrong) understanding is that there are actually a few situations where flexbox is recommended. (1) We want our elements aligned in a certain way or (2) there are elements that need to be in the same row or column, that without flexbox, we would need to float.

JavaScript DOM

by: Hary Potter
JavaScript Document Object Model (DOM) - allows programs and scripts to dynamically access and update the content, structure and style of a document. Finding HTML elements - document.getElementById ("id"); -> would only return one element since IDs should be unique. - document.getElementsByClassName ("class"); -> would need to use loop since it might return more than one element; all elements with that classname - document.getElementsByTagName ("tag"); -> would need to use loop since it might return more than one element; all elements with that tag Changing Attributes - you can change attributes of tags by directly accessing that attribute - e.g. document.getElementById ("icon").src = "new.jpg"; Changing Styles - you can change the style by accessing the style attribute - e.g. document.getElementById ("icon").style.color = red;

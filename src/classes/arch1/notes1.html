Positional Number System

by: Alexander Reyes
Here's the continuation for today's lesson! radix/base - the total # of unique symbols available in that system. - the largest-valued symbol is always radix - 1 (e.g. if decimal, radix = 10 so 10 - 1 = 9) Number Systems 1. Denary (or Decimal) Number System -> radix = 10 2. Binary Number System -> radix = 2 3. Octal Number System -> radix = 8 4. Hexadecimal Number System -> radix = 16 -> 0 to 9, A to F TRIVIA: Roman Numeral System is NOT positional number system. The value of its symbol is not based on its "position" in the number.

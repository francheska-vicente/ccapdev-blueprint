Von Neumann Architecture

by: Alexander Reyes
Hello, guys! This is my notes from today's lesson. Study well for the long test! Von Neumann Architecture - is a type of computer architecture that has three parts (or "black boxes"): (1) the memory, the CPU and (3) the I/O (input/output). - the architecture that is used by most computers nowadays - "store program architecture" -> the program and data are stored in the main memory and not in the CPU. - instructions in the main memory are fetched, decoded, and executed sequentially. - has three buses: (1) address bus, (2) data bus, and (3) control bus. - control bus -> used to carry control signals (what the "box" needs to do with the given data) -> originates from the CPU (as the CPU holds the instructions) - address bus -> uni-directional from CPU to memory or I/O - data bus -> used to transfer data to/from the 3 boxes
